06 Rules — القواعد
Framework

Next.js 15 — App Router · next/font/google للخطوط · next/font/local لـ Geist

CSS

Tailwind CSS · CSS Variables في globals.css · لا styled-components

الاتجاه

dir="rtl" · lang="ar" على <html> · Tailwind rtl: variants

الأيقونات

Lucide React — 16px inline · 20px واجهة · ممنوع إيموجي

الظلال

لا shadow-xl — الـ glass border تكفي — shadow-sm فقط إن لزم

الحركة

transition: .15s ease للـ hover · Framer Motion للـ page transitions · لا مبالغة

07 Design Principle
Core Principle
الوضوح أولاً.
كل عنصر له سبب.
